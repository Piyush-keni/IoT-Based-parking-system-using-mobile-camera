Project Description

An IoT-based Smart Parking System using Arduino, LDR sensors for slot detection, and a Servo SG90 motor for automated gate control. Customers book slots online; a Python + EasyOCR system reads number plates and verifies bookings via Google Sheets before granting gate access.
